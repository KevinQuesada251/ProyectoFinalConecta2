# 🗺️ Aplicativo Web de Mapa Interactivo y Comunidad

Este proyecto es una aplicación web que permite a los usuarios registrar ubicaciones en un mapa, interactuar en una comunidad, y acceder a funcionalidades personalizadas según su rol (usuario o administrador).

## 🚀 Funcionalidades Principales

### 👤 Funcionalidades para el Usuario

- 📍 Registro de Ubicaciones:
  - Los usuarios pueden hacer clic sobre un mapa para registrar nuevas ubicaciones.
  - Cada ubicación incluye un nombre, una descripción y coordenadas geográficas.
  - Las ubicaciones registradas se muestran en el perfil del usuario.

- 🧑‍💼 Perfil Personal:
  - Vista de ubicaciones registradas.
  - Cambio de foto de perfil y banner.
  - Interacción en el foro: comentar y responder a otros usuarios.

- 🗣️ Foro Comunitario:
  - Espacio para discusión, sugerencias y retroalimentación.
  - Comentarios y respuestas entre usuarios.

- 📢 Anuncios Oficiales:
  - Visualización de comunicados oficiales como actualizaciones, mantenimientos y nuevas funcionalidades.

- 📬 Contacto:
  - Formulario funcional de contacto integrado con EmailJS para enviar mensajes directamente al equipo del proyecto.

### 🛠️ Funcionalidades para Administradores

- 👮 Gestión de Usuarios:
  - Desactivar (banear) usuarios: el usuario no puede iniciar sesión aunque ingrese sus credenciales correctamente.
  - Editar la información de cualquier usuario.

- 📢 Gestión de Anuncios y Roles:
  - Crear, editar y eliminar anuncios oficiales.
  - Crear y administrar roles de usuario.

- 📦 Operaciones HTTP:
  - Las distintas secciones del panel de administración permiten ejecutar operaciones CRUD sobre los datos gestionados por el sistema.

### 🏠 Otras Secciones

- Página Home:
  - Página informativa que presenta la idea principal del proyecto.

- Página de Contacto:
  - Formulario funcional para enviar consultas vía EmailJS.

## 🧱 Tecnologías Usadas

- Frontend: React, CSS, Bootstrap
- Backend: Django
- Base de datos: MySQL
- Servicios externos: EmailJS
